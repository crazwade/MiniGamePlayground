# 簡介

minigameplayground 是使用 Vue 3 開發的迷你遊戲合集項目。它包含了多個經典迷你遊戲，例如貪吃蛇、1A2B、糖果消除等，讓您在閒暇之餘享受歡樂時光。

# 技術棧

- **框架：** Vue 3
  - **狀態管理：** Pinia
  - **路由：** Vue Router
- **樣式：**
  - **CSS 預處理器：** Sass/Less (可選)
  - **CSS Framework：** Tailwind CSS / Bootstrap (可選)
- **構建工具：** Vite
- **測試：** Vitest + cypress
- **樣式檢查：** ESLint + Prettier
- **Git：** 使用 GitFlow 工作流程管理代碼
